자동차 경주

---

## 기능목록

>게임 진행

- 이름을 입력한다
  - 이름은 쉼표로 구분한다
  - 5자 이하의 비어있지 않은 이름만 가능하다
  - 중복된 이름을 가지지 않는다
- 몇 라운드 진행  할지 입력한다
  - 1 이상의 정수만 입력받는다
- 진행 과정을 출력한다
  - 매 라운드 자동차 이름과 진행 거리를 보여준다
- 우승자를 출력한다
  - 가장 멀리 간 자동차가 우승
  - 우승자는 복수 가능

> 자동차

- 매 라운드 전진또는 멈춘다
  - 0 ~ 9사이의 랜덤값을 구한다
  - 4 이상이면 전진한다
  - 3 이하이면 멈춘다
