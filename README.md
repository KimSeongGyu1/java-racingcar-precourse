자동차 경주

---

구현할 기능 목록

- [입력] 
  - 자동차 이름 입력
    - 쉼표로 이름을 구분한다
    - 이름은 5자 이하만 가능하다
    - 이름은 중복되지 않는다.
  - 라운드 횟수 입력
    - 라운드 회수는 1 이상의 숫자만 입력받는다
  
- [자동차 동작]
  
  - 주어진 횟수 동안 각 자동차는 전진/멈춤
    - 전진 조건은 0-9사이의 random 값을 구한 후 4이상이면 전진한다
    - 멈춤 조건은 3 이하면 멈춘다
  
  - 각 라운드 마다 자기 이름과 현재 위치를 출력한다
  
- [경기 내용]
  
  - 라운드 별로 자동차 이름과 전진 거리를 출력한다
  
- [결과]
  - 가장 멀리 간 자동차가 우승
  - 우승자는 여러명 가능
